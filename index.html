Bitcoin Node Observer

Helm Chart for Bitcoin Stack

Welcome to the Bitcoin Node Observer Helm charts repository! This page provides the Bitcoin Stack chart that simplifies the deployment of Bitcoin node and Bitcoin exporter.

Bitcoin Stack Overview

The Bitcoin Stack chart is an umbrella chart that deploys the necessary components to run a Bitcoin signet node and monitor its metrics using an exporter. The chart includes:

  • Bitcoin Signet Node
  • BTC Exporter for node performance

How to Install

To get started, add this repository to your Helm configuration:

helm repo add bitcoin-node-observer https://aminechakr.github.io/bitcoin-node-observer/
helm repo update

Then, you can install the Bitcoin Stack chart:

helm install  bitcoin-node-observer/bitcoin-stack

Or, pull the package locally:

helm pull bitcoin-node-observer/bitcoin-stack --untar
